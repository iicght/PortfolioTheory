# Оптимизация инвестиционного портфеля инвестора на основе портфельной теории Марковица 

Проект демонстрирует применение портфельной теории Марковица для формирования оптимального портфеля активов по критерию максимизации коэффициента Шарпа.
Изначально разрабатывался в качестве курсовой работы. 

> Возможна дальнейшая доработка с поиском точек максимизации прибыли и минимизации риска
***
<li> Функциональность:

* Загрузка исторических данных по акциям через API _yfinance_ (Настройка временного интервала в файле main.py)
* Формирование месячных цен закрытия (Поскольку итоговый прогноз будет выдавать месячную доходность портфеля)
* Расчёт логарифмических доходностей (Можно использовать арифметическую в зависимости от объема выборки)
* Вычисление средней доходности и стандартного отклонения
* Построение ковариационной матрицы
* Генерация случайных портфелей (распределение Дирихле) с использованием _numpy_
* Расчёт коэффициента Шарпа
* Определение оптимального портфеля
* Построение графика и отображение на нем линии Шарпа при помощи _mathplotlib_
***


Структура проекта:
```
├── main.py              # Основной исполняемый модуль
├── data_loader.py       # Загрузка и подготовка данных
├── statistics.py        # Расчёт доходности и риска
├── portfolio.py         # Оптимизация портфеля
└── visualization.py     # Визуализация результатов
```
***


Технология  | Назначение
----------------|----------------------
Python| Язык программирования проекта
NumPy| Работа с данными, генерация случайных величин с заданным распределением
Pandas| Хранения данных с их дальнейшей обработкой
Matplotlib| Визуализация полученных данных
yfinance| Загрузка исторических данных с ценами указанных пользователем акций

***



После запуска необходимо ввести тикеры активов (Например AAPL, TSLA) и количество генерируемых портфелей. После этого на экран будет выведен график совокупности сгенерированных портфелей с проведенной прямой согласно коэффиценту Шарпа до оптимального портфеля. Далее программа выводит характеристики оптимального портфеля (Вес активов в портфеле и прогнозные значения).

### Запуск проекта
```
python main.py
```
